模式说明

  • 上帝后台：无需玩家输入，自动推进游戏，右侧显示全局监控信息。

  • 测试阶段玩家等待时间为 1 秒，超时会自动采用 Agent 建议。
  • 结束后会进入复盘页面，显示胜负结果和复盘状态。
