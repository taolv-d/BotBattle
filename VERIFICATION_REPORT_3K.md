## 修复的 Bug 清单

在验证过程中发现并修复了以下 bug：

### Bug 1: `_run_turn` 方法参数错误
- **严重程度**: 致命
- **问题**: 方法定义有 `player` 参数但调用时未传参
- **修复**: 移除多余参数，方法内部从 `self.current_player_id` 获取玩家
- **位置**: `games/threekingdoms/engine.py:268`

### Bug 2: 缺失技能方法
- **严重程度**: 严重
- **问题**: `_skill_sunquan_zhi` 和 `_skill_huanggai_ku` 方法未定义
- **修复**: 添加两个技能方法实现
- **位置**: `games/threekingdoms/engine.py:1232-1280`

### Bug 3: 决斗锦囊逻辑错误
- **严重程度**: 严重
- **问题**: 从源玩家手牌中移除目标的杀
- **修复**: 修正逻辑，正确从对应玩家手牌移除杀
- **位置**: `games/threekingdoms/engine.py:1071-1103`

### Bug 4: 玩家缺少 agent 属性
- **严重程度**: 致命
- **问题**: `ThreeKingdomsPlayer` 类没有 `agent` 属性，濒死求桃时崩溃
- **修复**: 
  1. 在 `ThreeKingdomsPlayer` 类添加 `agent: Optional[object] = None` 属性
  2. 在引擎 `setup` 方法中为 AI 玩家初始化 `ThreeKingdomsAgent`
- **位置**: `games/threekingdoms/state.py:234`, `games/threekingdoms/engine.py:200-223`
