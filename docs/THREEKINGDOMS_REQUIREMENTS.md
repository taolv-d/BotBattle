# 三国杀扩展 - 需求决策总结

**确认日期**: 2026-03-05  
**状态**: ✅ 已确认

---

## 核心决策

### 1. 游戏规则

| 问题 | 决策 | 说明 |
|------|------|------|
| 身份公开 | ✅ 死亡时立即公开 | 可配置，默认 true |
| 武将选择 | ✅ 随机 3 选 1 | 武将不能重复 |
| 距离计算 | ✅ 完整规则 | 不简化 |
| 濒死求桃 | ✅ 完整流程 | 逆时针询问 |
| 胜负判定 | ✅ 简化规则 | 主忠赢：反贼内奸全死；反贼赢：主公死；内奸赢：最后存活 |
| 游戏结束 | ✅ 即时判定 | 主公死立即结束 |
| 牌堆管理 | ✅ 固定牌堆 + 弃牌堆循环 | 不会平局 |

### 2. AI 设计

| 问题 | 决策 | 说明 |
|------|------|------|
| 思考记录 | ✅ 完全记录 | 方便调试 |
| 身份认知 | ✅ 只知道自己的身份 | 上帝视角玩家知道全部 |
| 记忆范围 | ✅ 必须 + 可选 | 记自己的牌 + 别人对自己出的牌 + 牌堆估计 |

### 3. UI 设计

| 问题 | 决策 | 说明 |
|------|------|------|
| 手牌展示 | ✅ 简单模式 | 只显示牌名 |
| 思考展示 | ✅ 可折叠/展开 | 默认显示最新，可展开看历史 |
| 玩家布局 | ✅ 线性布局 | 当前回合高亮 |
| 伤害追踪 | ✅ 简化 | 谁出的牌谁负责 |

### 4. 技术实现

| 问题 | 决策 | 说明 |
|------|------|------|
| 卡牌结构 | ✅ 数据类 | 类型安全 |
| 牌堆管理 | ✅ 固定牌堆 | 弃牌堆洗入循环 |
