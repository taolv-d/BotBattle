## 关键发现

### 已测试确认的致命 Bug

#### Bug 1: 预言家查验结果显示错误 ❌
- **现象**: 查验女巫时显示"witch"而不是"好人"
- **影响**: 上帝视角会看到具体角色名，破坏游戏平衡
- **位置**: `core/game_engine.py:419`
- **修复**: 将查验结果转换为二元值（好人/狼人）

#### Bug 2: 女巫可以毒杀自己 ❌
- **现象**: 测试中女巫（2 号）毒杀了 2 号（自己）
- **影响**: 严重逻辑错误，违反游戏规则
- **位置**: `core/game_engine.py:330-345`
- **修复**: 添加目标验证，排除自己

#### Bug 3: 猎人技能发动时目标列表包含自己 ❌
- **现象**: 猎人死亡后，存活玩家列表仍包含自己
- **影响**: 猎人可能带走自己（逻辑错误）
- **位置**: `core/game_engine.py:480`
- **修复**: 过滤掉猎人自己
