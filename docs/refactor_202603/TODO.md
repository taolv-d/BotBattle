## 🆕 新增功能（2026-03-20）

### 游戏复盘服务

**优先级**: P0
**预计工时**: 14h
**设计文档**: [GAME_REVIEW_DESIGN.md](GAME_REVIEW_DESIGN.md)

#### 功能描述

在游戏结束后自动调用 LLM 生成复盘报告，并支持检测对话中的逻辑漏洞。

#### 特性

- **通用模块**: 可被狼人杀、三国杀等游戏复用
- **双模式**: 复盘模式 + 分析模式（漏洞检测）
- **可配置**: 支持配置启用、LLM 选择、报告详细程度

#### 任务分解

- [ ] 实现 `GameReviewService` 基础框架
- [ ] 实现复盘报告生成 Prompt
- [ ] 实现漏洞检测 Prompt
- [ ] 集成到狼人杀游戏（`WerewolfOrchestrator`）
- [ ] 集成到三国杀游戏（`ThreeKingdomsOrchestrator`）
- [ ] 编写测试用例
- [ ] 编写文档

#### 输出示例

1. **复盘报告**: 游戏概览、关键事件、玩家表现、精彩时刻
2. **漏洞检测**: 信息泄露、逻辑矛盾、身份矛盾
