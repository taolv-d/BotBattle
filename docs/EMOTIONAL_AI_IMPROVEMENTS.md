# 狼人杀情感化改进

**改进日期**: 2026-03-05  
**改进内容**: 增强 AI 情感表达，让角色更有血有肉

---

## 问题描述

### 原有问题
AI 发言过于机械，缺少：
- 情感波动和情绪反应
- 口语化表达和语气词
- 内心纠结和复杂情感
- 对历史事件的情绪记忆

### 改进目标
让 AI 成为"有血有肉"的玩家：
- ✅ 被冤枉时会委屈、会着急
- ✅ 找到目标时会兴奋、会激动
- ✅ 撒谎时会紧张、被识破时会慌张
- ✅ 会犹豫、会纠结、会后悔

---

## 解决方案

### 1. 新增情感化系统提示

**文件**: `ai/agent.py`

**新增方法**: `_build_emotional_system_prompt()`

```python
def _build_emotional_system_prompt(self) -> str:
    """构建情感丰富的系统提示 - 让 AI 更有血有肉"""
    
    # 角色情感化描述
    role_descriptions = {
        Role.WEREWOLF: """🐺 你是狼人 - 但要演得像好人！
- 内心紧张但要表面镇定
- 要会撒谎，但不能太明显
- 看到队友被怀疑时要会救""",
        Role.SEER: """🔮 你是预言家 - 压力山大！
- 手握真相但要小心暴露
- 被质疑时会委屈、会着急
- 找到狼人时会兴奋、会激动""",
        # ...
    }
    
    # 人格情感化描述
    personality_emotions = {
        "honest": "你是个诚实的人，被冤枉时会委屈，发现谎言时会愤怒",
        "liar": "你擅长撒谎，骗人时内心窃喜，被识破时会慌张",
        "aggressive": "你脾气火爆，被质疑时会暴怒，找到目标时会兴奋",
        # ...
    }
    
    return f"""你是一个富有情感、有血有肉的狼人杀玩家。

【你的性格】{personality_name}
{emotion_hint}

【发言要求】
1. 性格驱动：发言必须符合你的性格设定，有明确的情感倾向
2. 内心独白：输出真实动机和情感波动
3. 人情味：使用口语化表达，加入语气词
4. 历史记忆：结合游戏历史中的具体事件展现情绪反应
5. 情绪波动：根据局势变化展现各种情绪
"""
```

### 2. 增强发言上下文

**改进内容**:
- 添加历史记忆（其他玩家说过的话）
- 添加夜晚死亡情感提示
- 完整显示发言内容（不截断）

```python
def speak(self, context: dict, round_num: int = 1):
    # 添加记忆中的关键事件（情感化）
    memory_hint = ""
    recent_events = [m for m in self.memory[-10:] if m.get("type") == "speech"]
    if recent_events:
        events_desc = []
        for event in recent_events:
            if event.get("player_id") != self.player.id:
                events_desc.append(f"{event.get('player_id')}号曾在第{event.get('round', 1)}轮说过：{event.get('content', '')[:40]}")
        memory_hint = "\n历史记忆:\n" + "\n".join(events_desc)
    
    # 使用情感化系统提示
    system_prompt = self._build_emotional_system_prompt()
```

### 3. 情感化发言要求

```
【情感表达要求】
- 使用口语化表达，加入"啊"、"吧"、"呢"、"我觉得"、"说实话"等语气词
- 展现真实情感：紧张、兴奋、委屈、愤怒、疑惑等
- 如果有玩家踩你/保你，要表现出相应的情绪反应
- 内心独白要真实，可以包含犹豫、纠结、窃喜等复杂情绪
```

## 后续优化建议

1. **更细致的情感分类**
   - 添加更多情绪类型：惊讶、失望、无奈等
   - 根据局势动态调整情绪强度

2. **长期记忆**
   - 记住整局游戏中发生的关键事件
   - 对之前的误会、恩怨做出情感反应

3. **表情和动作**
   - 添加简单的表情描述：[皱眉]、[冷笑]、[摊手]
   - 增加游戏的代入感

4. **个性化表达**
   - 每个 AI 有独特的口头禅
   - 不同人格使用不同的语气词
